## Problema 1 [2 pts]

Suponer que la variable $X$ tiene distribución exponencial parametrizada por la esperanza:

$$
p(x\mid\lambda) =
\begin{cases}
\dfrac{1}{\lambda}\exp\left(-\dfrac{x}{\lambda}\right) & \text{si } x \ge 0,\\
0 & \text{en caso contrario.}
\end{cases}
$$

- **a)** Suponer que $n$ ejemplos $x_1, \cdots, x_n$ se generan independientemente de acuerdo a $p(x\mid\lambda)$. Mostrar que el estimador de máxima verosimilitud para $\lambda$ está dado por

$$
\hat{\lambda} = \overline{X} = \frac{1}{n}\sum_{i=1}^{n} X_i.
$$

Suponer a continuación que se dispone de dos muestras de entrenamiento, correspondientes a cada una de las poblaciones.

- **b)** Describir al menos dos reglas de clasificación para dos poblaciones exponenciales con medias desconocidas.

## Problema 2 [2 pts]

Las distribuciones condicionales para un problema unidimensional con dos categorías son distribuciones de Cauchy:

$$
p(x\mid\omega_1) = \frac{1}{5\pi}\frac{1}{1+\left(\frac{x}{5}\right)^2},
\qquad
p(x\mid\omega_2) = \frac{1}{5\pi}\frac{1}{1+\left(\frac{x-8}{5}\right)^2}.
$$

- **a)** Suponner que $P(\omega_1) = P(\omega_2)$ y notar que $P(\omega_1\mid x) = P(\omega_2\mid x)$ si $x = 4$. Construir el clasificador de Bayes para este problema.
- **b)** Simular una muestra de test de ambas distribuciones con 1000 datos cada una, ayudado por el siguiente código:

```python

from scipy.stats import cauchy

# muestra de omega 1 que tiene posicion = 0 y escala = 5
cauchy.rvs(loc=0, scale=5, size=1000)

# muestra de omega 2 que tiene posicion = 8 y escala = 5
cauchy.rvs(loc=8, scale=5, size=1000)
```

y estimar numéricamente el error aparente de clasificación. Comparar con el error exacto de Bayes, que en este problema viene dado por

$$
P(\mathrm{error}) = \frac{1}{2} - \frac{1}{\pi}\arctan(4).
$$
